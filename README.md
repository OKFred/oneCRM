# Vite + Vue 3 + Vue Router + Ant Design + AntV G2 + Font Awesome

This is a template for CRM, compatible for both PC and mobile, and mutli-language support (i18n).

CRM模板，适配PC / mobile，支持多语言
包含【登录页，首页，客户管理，时间管理，业绩】五个主要页面。
